募金カウントWebアプリ

棒金を含めた金額計算ができるシンプルなWebアプリ

Web Vanilla JS UI Responsive

概要

入力された金額をもとに棒金処理を含めた総額を計算し、瞬時に結果を表示するWebアプリです。 インストール不要・ブラウザだけで動作し、スマホでも快適に利用できます。

画面イメージ

主な機能

  • 金額の入力と即時計算
  • 棒金を含めた合計金額の表示
  • PC・スマホどちらでも使いやすいレスポンシブデザイン

使用技術

  • HTML5
  • CSS3
  • Vanilla JavaScript
  • GitHub Pages デプロイ

工夫した点

  • レスポンシブ対応でスマホでも快適に操作
  • 計算が即時に反映されるスムーズなUI
  • インストール不要・軽量な設計
